# Crypto Trading Bot (SMA 5/20 + Stop-Loss 10%)

Автоматизированный торговый бот для криптовалют, использующий стратегию на основе скользящих средних.
Предназначен для **торговли на тестовых сетях** (Binance Testnet) с жестким стоп-лоссом для управления рисками.

---

## Стратегия

Бот реализует классическую стратегию пересечения скользящих средних:

| **Индикатор**     | **Период** |
| :---------------- | :--------- |
| **Быстрая SMA**   | 5 свечей   |
| **Медленная SMA** | 20 свечей  |

**Логика сигналов:**

- 🟢 **Покупка (BUY)**: Быстрая SMA (5) пересекает медленную SMA (20) **снизу вверх**.
- 🔴 **Продажа (SELL)**: Быстрая SMA (5) пересекает медленную SMA (20) **сверху вниз**.

**Управление рисками:**

- **Стоп-лосс**: Установлен на уровне **-10%** от цены входа. При срабатывании позиция закрывается.

## Полная инструкция по установке и запуску

### Требования

- **Python 3.8** или выше ([скачать](https://www.python.org/downloads/))
- Аккаунт на **Binance Testnet** ([регистрация](https://testnet.binance.vision/))
- Терминал / командная строка

---

### Шаг 1: Клонируй репозиторий

```bash
git clone https://github.com/ERazina/binance_test_bot.git
cd your-repo-name
```

### Шаг 2: Установи библиотеки

```bash
pip3 install -r requirements.txt
```

### Шаг 3: Создай файл .env и положи туда ключи BINANCE_API_KEY и BINANCE_API_SECRET, которые ты сам сгенерируешь на https://testnet.binance.vision/

### Шаг 4: Запусти бота

```bash
python3 bot.py
```
